# Pneumonia Detection using Deep Learning (CNN)

## Overview

Pneumonia Detection using Deep Learning is a medical image classification project developed to detect pneumonia from Chest X-ray images using a Convolutional Neural Network (CNN).

The model is trained to classify X-ray images into two categories:

* Normal
* Pneumonia

This project demonstrates the practical application of Deep Learning and Computer Vision in healthcare for assisting early pneumonia diagnosis and medical image analysis.



## Project Objectives

* Detect pneumonia from Chest X-ray images
* Build and train a CNN-based image classification model
* Improve medical diagnosis support using AI
* Perform image preprocessing and feature extraction
* Evaluate model performance on medical datasets
* Demonstrate healthcare applications of Deep Learning



## Features

* CNN-based image classification
* Chest X-ray image analysis
* Binary classification (Normal vs Pneumonia)
* Image preprocessing and normalization
* Model training and evaluation
* Prediction using trained Deep Learning model
* Medical AI application workflow



## Dataset

The dataset used in this project is:
